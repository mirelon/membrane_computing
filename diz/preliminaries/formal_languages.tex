Our study is based on the classical theory of formal languages. We will recall some definitions:

\begin{definition}
An \index{Alphabet} {\bf alphabet} is a finite nonempty set of symbols.
\end{definition}

\begin{definition}
A \index{String} {\bf string} over an alphabet is a finite sequence of symbols from the alphabet.
\end{definition}

The length of the string $s$ is denoted by $|s|$. We denote by $\Sigma^*$ the set of all strings over an alphabet $\Sigma$. By $\Sigma^+$ = $\Sigma^* - \{\eps\}$ we denote the set of all nonempty strings over $\Sigma$.

\begin{definition}
A \index{Language} {\bf language} over the alphabet $\Sigma$ is any subset of $\Sigma^*$.
\end{definition}

\begin{definition}
A \index{Language!family} {\bf family of languages} is a set of languages.
\end{definition}
